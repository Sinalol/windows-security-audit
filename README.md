# Windows Security Audit

A beginner PowerShell cybersecurity project that performs basic Windows security checks and creates a report on the user's Desktop.

## What It Checks

- Windows Defender status
- Firewall status
- Open TCP network connections
- Local user accounts
- Top running processes
- Recent failed login events

## Purpose

This project was created as part of my cybersecurity learning portfolio. It demonstrates basic Windows administration, PowerShell scripting, security monitoring, and system auditing skills.

## How to Run

1. Open PowerShell as Administrator.
2. Navigate to the folder containing the script.
3. Run:

```powershell
.\windows_security_audit.ps1
